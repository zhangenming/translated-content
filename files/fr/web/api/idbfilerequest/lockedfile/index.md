## Résumé

La propriété `lockedFile` représente l'objet {{domxref("LockedFile")}} à partir duquel la requête a été lancée.

## Syntaxe

    var lockedFile = instanceOfFileRequest.lockedFile

## Valeur

Un objet {{domxref("LockedFile")}}.
