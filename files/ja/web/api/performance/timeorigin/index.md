**`timeOrigin`** は {{domxref("Performance")}} インターフェイスの読み取り専用プロパティで、パフォーマンス測定の開始時刻の高解像度タイムスタンプを返します。

{{AvailableInWorkers}}

## 構文

```js
var timeOrigin = performance.timeOrigin
```

### 値

高解像度のタイムスタンプです。
