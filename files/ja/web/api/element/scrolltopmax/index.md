{{APIRef("DOM")}}{{Non-standard_header}}

**`Element.scrollTopMax`** は読み取り専用のプロパティで、この要素で可能な最大の上スクロールオフセットを表す数値を返します。

## 構文

```js
var pxl = elt.scrollTopMax;
```

## 仕様書

_このプロパティはどの仕様書にも含まれていません。_
