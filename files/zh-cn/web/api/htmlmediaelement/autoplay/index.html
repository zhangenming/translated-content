{{APIRef("HTML DOM")}}

HTMLMediaElement.autoplay 属性等同于HTML属性中的 {{htmlattrxref("autoplay", "video")}}, 它表示当player下载到足够可以播放的媒体文件时，是否自动播放视频。

注意: 一些版本的Chrome只支持autostart, 不支持autoplay
