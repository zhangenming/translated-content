{{APIRef}}

当一个被拖动的元素或者被选择的文本离开一个有效的拖放目标时，将会触发dragleave 事件。

基本信息

Bubbles Yes
Cancelable No
Target objects {{domxref("Document")}}, {{domxref("Element")}}
Interface {{domxref("DragEvent")}}
Default Action None.
