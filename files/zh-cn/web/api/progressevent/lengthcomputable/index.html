{{APIRef("DOM Events")}}

ProgressEvent.lengthComputable 只读属性是一个布尔{{domxref("Boolean")}} 标志，表示{{domxref("ProgressEvent")}} 所关联的资源是否具有可以计算的长度。否则 ，{{domxref("ProgressEvent.total")}} 属性将是一个无意义的值。

语法

flag = ProgressEvent.lengthComputable
