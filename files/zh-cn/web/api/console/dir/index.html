{{APIRef("Console API")}}{{Non-standard_header}}

在控制台中显示指定JavaScript对象的属性，并通过类似文件树样式的交互列表显示。

{{AvailableInWorkers}}

语法

console.dir(object);

参数

object
打印出该对象的所有属性和属性值.
