概述

在 Web控制台上创建一个新的分组.随后输出到控制台上的内容都会被添加一个缩进,表示该内容属于当前分组,直到调用{{ domxref("console.groupEnd()") }}之后,当前分组结束.

语法

console.group();

参数

无.

备注

在文档{{ domxref("console") }}中查看在控制台中使用分组,了解更多详细内容.

规范

不属于任何公开的规范
