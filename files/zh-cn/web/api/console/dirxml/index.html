{{APIRef("Console API")}}{{Non-standard_header}}

显示一个明确的XML/HTML元素的包括所有后代元素的交互树。 如果无法作为一个element被显示，那么会以JavaScript对象的形式作为替代。 它的输出是一个继承的扩展的节点列表，可以让你看到子节点的内容。

语法

console.dirxml(object);

参数

object
一个属性将被输出的JavaScript对象。
