{{ApiRef("URL API")}}
{{domxref("URL")}} 接口的端口属性是包含了URL的端口号信息的{{domxref("USVString")}}值，如果URL中不包含明确的端口号，这个属性将为''.

{{AvailableInWorkers}}

语法

string = object.port;
object.port = string;

参数

A {{domxref("USVString")}}.

示例

var url = new URL('https://mydomain.com:80/svn/Repos/');
var result = url.port; // Returns:'80'
