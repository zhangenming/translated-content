{{CSSRef}}

CSS Fonts 是一个CSS模块，它定义字体相关的属性和字体资源是如何加载的。它允许定义一个字体的样式，如字体样式、尺寸大小或字体宽度，以及要使用的字形的变体（加粗、斜体等），对于一个字符有几个字形的字体。
