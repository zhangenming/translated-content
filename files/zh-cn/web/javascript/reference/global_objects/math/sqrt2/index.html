概述

Math.SQRT2 属性表示 2 的平方根，约为 1.414：

描述

由于 SQRT2 是 Math 的静态属性，所以应该像这样使用：Math.SQRT2，而不是作为你创建的 Math 实例的属性（Math 不是构造函数）。

示例

例子：使用 Math.SQRT2

下面的函数返回 2 的平方根：

function getRoot2() {
   return Math.SQRT2;
}

getRoot2(); // 1.4142135623730951
