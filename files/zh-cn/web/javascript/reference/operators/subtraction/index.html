The subtraction operator (-) subtracts the two operands, producing their difference.

{{EmbedInteractiveExample("pages/js/expressions-subtraction.html")}}

语法

Operator: x - y

Examples

Subtraction with numbers

5 - 3     // 2
3 - 5     // -2

Subtraction with non-numbers

'foo' - 3 // NaN
