El operador de sustracción (-) sustrae dos operandos, produciendo su diferencia.

{{EmbedInteractiveExample("pages/js/expressions-subtraction.html")}}

Sintaxis

Operator: x - y

Ejemplos

Sustracción con números

5 - 3     // 2
3 - 5     // -2

Sustracción de no numéricos

'foo' - 3 // NaN
