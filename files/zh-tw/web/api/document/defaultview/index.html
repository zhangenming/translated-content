{{ApiRef}}

在瀏覽器中，document.defaultView 屬性會指向一個目前 {{Glossary("Browsing_context", "document")}} 所屬的 {{domxref("Window", "window")}} 物件，若無則為 null。

語法

var win = document.defaultView;

此為唯讀屬性。
